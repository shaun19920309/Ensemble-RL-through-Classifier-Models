# Holding-Disagreement Ablation

These CSV files are the final tables reported by the current paper. They cover
three metrics, three markets, three RL pairs, five classifier groups, 89 fixed
global thresholds, and 30 rolling classifier refits per configuration.

The exact configuration-level comparison figure is stored at
`../paper_figures/disagreement_metric_ablation/figure_v2_disagreement_ablation_all_markets.pdf`.
Full per-threshold daily traces are reproducible with the command in
`EXPERIMENTS.md` and are intentionally excluded as intermediate data.
